FIVE AGGREGATE FUNCTIONS IN SQL
Aggregate functions are built-in SQL functions that perform a calculation on a set of values and return a single result. They work on multiple rows and return one value.
Common aggregate functions: COUNT, SUM, AVG, MIN, MAX
AGGREGATE FUNCTION 1: COUNT()
Definition:
COUNT() returns the number of rows that match specified criteria. It counts the number of non-NULL values in a column or all rows in a table.
Syntax:
COUNT(*)           -- Counts all rows including NULLs
COUNT(column_name) -- Counts non-NULL values in column
COUNT(DISTINCT column_name) -- Counts unique non-NULL values
Characteristics:

Returns an integer value
COUNT(*) counts all rows
COUNT(column) ignores NULL values
Can be used with WHERE clause
Often combined with GROUP BY

Example 1: Count total employees
SELECT COUNT(*) AS TotalEmployees
FROM emp;
Output:
TotalEmployees
8
Example 2: Count employees in each department
SELECT dno, COUNT(*) AS EmployeeCount
FROM emp
GROUP BY dno;
Output:
dno    EmployeeCount

10     3
20     1
30     2
40     2
Example 3: Count employees with salary assigned
SELECT COUNT(salary) AS EmployeesWithSalary
FROM emp;
Output:
EmployeesWithSalary
8
Example 4: Count distinct departments
SELECT COUNT(DISTINCT dno) AS DepartmentCount
FROM emp;
Output:
DepartmentCount
4
Example 5: Count employees earning above 20000
SELECT COUNT(*) AS HighEarners
FROM emp
WHERE salary > 20000;
Output:
HighEarners
6
Real-world usage:

Total number of customers in database
Number of orders placed by specific customer
Count of products in inventory
Number of employees by department
Statistics and reporting

AGGREGATE FUNCTION 2: SUM()
Definition:
SUM() returns the total sum of numeric values in a column. It adds up all values and returns single result.
Syntax:
SUM(column_name)           -- Sum all values
SUM(DISTINCT column_name)  -- Sum unique values
SUM(column_name) WHERE condition -- Sum with condition
Characteristics:

Works only on numeric columns
Ignores NULL values
Returns NULL if no rows match
Can use with WHERE clause
Often used with GROUP BY

Example 1: Total salary of all employees
SELECT SUM(salary) AS TotalSalaryExpense
FROM emp;
Output:
TotalSalaryExpense
175000.00
Example 2: Total salary by department
SELECT dno, SUM(salary) AS DepartmentSalary
FROM emp
GROUP BY dno
ORDER BY DepartmentSalary DESC;
Output:
dno    DepartmentSalary

10     67000.00
40     47000.00
30     43000.00
20     18000.00
Example 3: Total sales for each product
SELECT product_id, product_name, SUM(quantity_sold) AS TotalQuantity
FROM sales
GROUP BY product_id, product_name
ORDER BY TotalQuantity DESC;
Output:
product_id    product_name      TotalQuantity

P001          Laptop            150
P002          Mobile Phone      200
P003          Tablet            120
Example 4: Sum with WHERE clause
SELECT SUM(salary) AS SalaryAbove20000
FROM emp
WHERE salary > 20000;
Output:
SalaryAbove20000
130000.00
Example 5: Total rent collected from tenants
SELECT SUM(rent_amount) AS TotalRentCollected
FROM rentals
WHERE rental_date BETWEEN '2026-02-01' AND '2026-02-28';
Output:
TotalRentCollected
45000.00
Real-world usage:

Total sales revenue
Total expenses by department
Total amount paid by customer
Total inventory value
Financial reporting and accounting

AGGREGATE FUNCTION 3: AVG()
Definition:
AVG() calculates the average (mean) value of numeric column. It sums all values and divides by count of non-NULL values.
Syntax:
AVG(column_name)           -- Average of all values
AVG(DISTINCT column_name)  -- Average of unique values
AVG(column_name) WHERE condition -- Average with condition
Characteristics:

Works only on numeric columns
Ignores NULL values
Returns decimal number
Useful for finding middle ground values
Often combined with GROUP BY and HAVING

Example 1: Average salary of all employees
SELECT AVG(salary) AS AverageSalary
FROM emp;
Output:
AverageSalary
21875.00
Example 2: Average salary by department
SELECT dno, AVG(salary) AS DepartmentAverageSalary
FROM emp
GROUP BY dno
ORDER BY DepartmentAverageSalary DESC;
Output:
dno    DepartmentAverageSalary

40     23500.00
10     22333.33
30     21500.00
20     18000.00
Example 3: Find departments with average salary above 22000
SELECT dno, AVG(salary) AS AverageSalary
FROM emp
GROUP BY dno
HAVING AVG(salary) > 22000;
Output:
dno    AverageSalary

10     22333.33
40     23500.00
Example 4: Average price of products in each category
SELECT category, AVG(price) AS AveragePrice
FROM product
GROUP BY category
ORDER BY AveragePrice DESC;
Output:
category      AveragePrice

Electronics   45000.00
Furniture     15000.00
Clothing      2500.00
Example 5: Average rating of movies
SELECT AVG(rating) AS AverageMovieRating
FROM movie
WHERE release_year = 2025;
Output:
AverageMovieRating
7.85
Real-world usage:

Average student score in class
Average product price in category
Average customer spending
Average employee salary
Average order value

AGGREGATE FUNCTION 4: MIN()
Definition:
MIN() returns the minimum (smallest) value in a column. It finds the lowest value among all rows.
Syntax:
MIN(column_name)           -- Minimum value in column
MIN(column_name) WHERE condition -- Minimum with condition
Characteristics:

Works on numeric, date, and string columns
Ignores NULL values
Returns single value
String comparison is alphabetically
Can be used with GROUP BY

Example 1: Lowest salary in company
SELECT MIN(salary) AS LowestSalary
FROM emp;
Output:
LowestSalary
18000.00
Example 2: Minimum salary in each department
SELECT dno, MIN(salary) AS MinimumSalary
FROM emp
GROUP BY dno;
Output:
dno    MinimumSalary

10     20000.00
20     18000.00
30     19000.00
40     21000.00
Example 3: Youngest employee

FROM emp;
Output:

(Oldest emp) 1995-03-25
Example 4: Cheapest product in each category
SELECT category, product_name, MIN(price) AS MinPrice
FROM product
GROUP BY category;
Output:
category       product_name         MinPrice

Electronics    Budget Phone         15000.00
Furniture      Simple Chair         5000.00
Clothing       Basic T-shirt        500.00
Example 5: Earliest order date
SELECT MIN(order_date) AS FirstOrderDate
FROM orders;
Output:

Real-world usage:

Find youngest/oldest employee
Cheapest product price
Earliest transaction date
Minimum temperature recorded
Lowest score in exam

AGGREGATE FUNCTION 5: MAX()

Syntax:
MAX(column_name)           -- Maximum value in column
MAX(column_name) WHERE condition -- Maximum with condition

Works on numeric, date, and string columns
Ignores NULL values
Returns single value
String comparison is alphabetically last
Can be used with GROUP BY

COMBINING MULTIPLE AGGREGATE FUNCTIONS
Example: Complete statistics for each department
SELECT
dno,
COUNT(*) AS EmployeeCount,
SUM(salary) AS TotalSalary,
AVG(salary) AS AverageSalary,
MIN(salary) AS MinimumSalary,
MAX(salary) AS MaximumSalary
FROM emp
GROUP BY dno
ORDER BY dno;
Output:
dno    Count  Total    Average   Minimum   Maximum

10     3      67000    22333.33  20000     25000
20     1      18000    18000.00  18000     18000
30     2      43000    21500.00  19000     24000
40     2      47000    23500.00  21000     26000

AGGREGATE FUNCTIONS WITH WHERE CLAUSE
Example: February sales analysis
SELECT
COUNT(*) AS TotalTransactions,
SUM(quantity * price) AS TotalRevenue,
AVG(quantity * price) AS AverageTransactionValue,
MIN(price) AS CheapestItemPrice,
MAX(price) AS MostExpensiveItemPrice
FROM sales
WHERE MONTH(sale_date) = 2 AND YEAR(sale_date) = 2026;
Output:
Transactions  Revenue   Average    Cheapest  Expensive

5            480000    96000.00    20000     50000
AGGREGATE FUNCTIONS WITH HAVING CLAUSE
Example: Find products with more than 2 sales
SELECT
product_name,
COUNT() AS SalesCount,
SUM(quantity) AS TotalQuantity,
AVG(price) AS AvgPrice
FROM sales
GROUP BY product_name
HAVING COUNT() > 1;
Output:
product_name  SalesCount  TotalQuantity  AvgPrice

Laptop       2           3                50000.00
Mobile       2           9                30000.00
SUMMARY TABLE OF FIVE AGGREGATE FUNCTIONS
Function  Input           Output           Use Case                    Example

COUNT()   All types       Integer          Count rows/records          COUNT(*) returns 8
SUM()     Numeric only    Numeric          Total amount                SUM(salary) = 175000
AVG()     Numeric only    Decimal          Average value               AVG(salary) = 21875
MIN()     All types       Same type        Minimum/lowest              MIN(salary) = 18000
MAX()     All types       Same type        Maximum/highest             MAX(salary) = 26000
Rules for aggregate functions:

Must use GROUP BY with other columns in SELECT
Cannot use WHERE to filter aggregates (use HAVING instead)
NULL values are ignored by COUNT, SUM, AVG, MIN, MAX
COUNT(*) counts all rows including NULLs
Cannot nest aggregate functions: SUM(AVG(salary)) is invalid
Works with numeric, date, and string columns (varies by function)
